# Astrolio - Adnan's Digital Self-Portrait

A minimalist, high-performance "Digital Garden" and portfolio built with **Astro 5** and **Tailwind CSS**. This isn't just a portfolio; it's a place where I document my journey, struggles, and technical explorations with honesty.

## 🚀 Key Features

- **Linux Commands Wiki**: A dedicated, type-safe space for my personal Linux command reference and hacks.
- **Milestones Log**: Documenting my personal growth, reflections, and lessons learned—acknowledging the mess and celebrating progress.
- **Focus Dashboard**: Real-time progress tracking of my current projects with a `pacman` (Arch Linux package manager) style UI.
- **Atmospheric Visuals**: A custom-designed animated night sky background with twinkling stars and subtle shooting stars, featuring `prefers-reduced-motion` support.
- **Production-Ready SSR**: Optimized for Cloudflare Pages with Server-Side Rendering (SSR) and smart pre-rendering for performance.
- **OLED Optimized**: Absolute black (`#050505`) background with sharp **Cyan** accents for a premium developer aesthetic.

## 🛠 Tech Stack

- **Framework**: [Astro 5](https://astro.build/) (Static Site Generation + SSR)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with Typography plugin.
- **Language**: [TypeScript](https://www.typescriptlang.org/) for data integrity and type-safety.
- **Deployment**: [Cloudflare Pages](https://pages.cloudflare.com/) with custom Cloudflare adapter.
- **Asset Management**: [Git LFS](https://git-lfs.github.com/) for high-quality WebP images.

## 📦 Getting Started

Ensure you have **Node.js** and **pnpm** installed.

```bash
# Clone the repository
git clone https://github.com/aadnanmt/astrolio.git

# Install dependencies
pnpm install

# Start development server
pnpm dev
```
